##💼 StyleMate — 패션 커뮤니티 & 가상 피팅 플랫폼

StyleMate는 사용자가 코디를 공유하고, 아바타를 통해 착용 모습을 미리 확인할 수 있는 웹 서비스입니다.
패션 피드, 피팅룸, 관리자 기능까지 한 곳에서 다룰 수 있도록 구성했습니다.

📌 기능 소개

● 사용자  
- 회원가입 / 로그인  
- 프로필 이미지 및 정보 수정  
- 내가 작성한 피드, 좋아요한 피드 모아보기  

● 피드  
- 글 + 이미지 등록, 수정, 삭제  
- 좋아요, 댓글/대댓글  
- 태그 기반 검색, 최신·인기 정렬  

● 관리자 페이지  
- 회원 목록/권한 관리  
- 피드/댓글 삭제 처리  
- 간단한 통계 확인 화면  

● 가상 피팅룸  
- 신체 정보 기반 아바타 생성  
- 의류 이미지 착용/해제  
- 룩 저장 기능  
- Python 크롭 서버로 배경 제거  

🛠 기술 스택  
Backend: Java17, Spring Boot, Spring Security, JWT, JPA, MySQL  
Frontend: Thymeleaf, Tailwind CSS, JS  
기타: Gradle, Flask 이미지 크롭 서버

🧱 프로젝트 구조  
- Controller → Service → Repository  
- DTO/Entity 분리  
- 공통 예외 처리  
- 이미지 업로드 로컬/S3 선택 가능  

📂 화면 흐름  
- 메인 피드: 최신/인기/태그  
- 피드 상세: 이미지 · 댓글/대댓글  
- 마이페이지: 내 정보, 내 피드  
- 관리자: 유저/게시물 관리  
- 피팅룸: 사이즈 입력 → 아바타 생성 → 아이템 착용/저장
