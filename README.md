# StyleMate 👕✨

💼 StyleMate — 패션 커뮤니티 & 가상 피팅 플랫폼

StyleMate는 사용자가 코디를 공유하고, 아바타를 통해 착용 모습을 미리 확인할 수 있는 웹 서비스입니다.
패션 피드, 피팅룸, 관리자 기능까지 한 곳에서 다룰 수 있도록 구성했습니다.

📌 기능 소개
● 사용자

회원가입 / 로그인

프로필 이미지 및 정보 수정

내가 작성한 피드, 좋아요한 피드 모아보기

● 피드

글 + 이미지 등록, 수정, 삭제

좋아요, 댓글/대댓글

태그 기반 검색, 최신·인기 정렬

● 관리자 페이지

회원 목록/권한 관리

피드/댓글 삭제 처리

간단한 통계 확인 화면

● 가상 피팅룸

사용자가 입력한 신체 정보 기반 아바타 생성

의류 이미지 착용/해제

룩 저장 기능

별도 Python 크롭 서버로 배경 제거 처리

🛠 기술 스택

Backend

Java 17, Spring Boot

Spring Security, JWT

Spring Data JPA

MySQL

Frontend

Thymeleaf

Tailwind CSS

JavaScript

Gradle

Python(Flask) 기반 크롭 서버

🧱 프로젝트 구조

Controller → Service → Repository 구조

Entity와 DTO 분리

공통 예외 처리 핸들러 적용

이미지 업로드 부분은 로컬/S3 중 선택 가능하도록 분리

📂 화면 흐름 요약

메인/피드 목록: 최신순, 인기순, 태그 필터

피드 상세: 이미지 슬라이드, 댓글/대댓글

마이페이지: 기본 정보 수정, 작성 글/좋아요 모아보기

관리자: 사용자 관리, 신고 내역, 게시글 제재

피팅룸: 사이즈 입력 → 아바타 생성 → 아이템 착용/저장
